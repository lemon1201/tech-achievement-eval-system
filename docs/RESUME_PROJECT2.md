# 项目2简历描述（可直接粘贴）

## 版本A（通用后端/AI应用岗）
**科技成果评价辅助系统（FastAPI）**

- 负责搭建科技成果评价的端到端服务，抽象“任务分类-案例召回-模块提取-AHP选择-大纲生成-一致性校验-文档导出”流程，并以 API 形式对外提供。
- 设计适用边界过滤机制（行业、时限、预算、合规等级），将规则前置到模块提取阶段，提升候选模块相关性与可解释性。
- 实现阈值冲突检测，识别大纲中互斥/不一致条件，支持在生成与校验阶段输出冲突标记，降低人工复核成本。
- 打通交付链路，支持 Word/PDF 大纲导出，提升从“模型结果”到“评审材料”的落地效率。

## 版本B（偏工程化表达）
**科技成果评价辅助系统｜Python + FastAPI + Pydantic**

- 基于分层架构拆分 `api / services / models / storage`，沉淀可复用服务模块，降低耦合并提升后续扩展效率。
- 建立 `/pipeline/run` 聚合接口，将多阶段能力编排为单次调用，统一输入/输出协议。
- 为大纲生成链路增加规则一致性校验与冲突标签输出，提高结果可审计性。
- 增强导出能力（Word/PDF）及文档化说明（README + TECH_DESIGN），支持面试演示与快速复现。
